Web Design

1. O que é Web Design e qual sua relação com Front-End?

Resposta:

O web design é a parte visual e estética de um website, envolvendo a criação da identidade visual, definição de paletas de cores, escolha de fontes, criação de layouts e disposição dos elementos na página. O front-end é a parte do desenvolvimento web que lida com a interface do usuário e a interação com o website, desenvolvendo o código HTML, CSS e JavaScript necessários para tornar o design visualmente atraente e funcional

2. Cite três princípios de design que melhoram a experiência do usuário.

Resposta:

  • 1: Clareza e simplicidade Interfaces claras e simples permitem que os usuários compreendam rapidamente como interagir com o produto. Evitar sobrecarga visual, excesso de informações ou elementos desnecessários ajuda a reduzir erros e frustração. Um design simples prioriza as funções essenciais e apresenta ação e informação de forma direta, facilitando a tomada de decisão pelo usuário.
  • 2: Consistência Manter consistência visual e funcional é crucial para que os usuários aprendam rapidamente a navegar pela interface. Isso inclui padrões de cores, tipografia, ícones, posicionamento de botões e comportamento de elementos interativos.
  • 3: Foco no Usuário Colocar o usuário no centro do processo de design, compreendendo suas necessidades, preferências e limitações.

3. Qual a diferença entre Wireframe e Protótipo?

Resposta: Wireframe Representação básica e funcional do layout do projeto, mostrando a estrutura da interface sem focar em aspectos estéticos

resposta: Protótipo Versão interativa do design, simula a navegação entre as telas, os cliques, as transições e o comportamento da interface como se estivesse “funcionando”, mesmo sem código

6. Qual a diferença entre uma tag em bloco e uma tag inline?

Resposta: Elementos de bloco ocupam toda a largura disponível e começam em novas linhas, enquanto elementos em linha ocupam apenas o espaço necessário e não começam em novas linhas. Elementos de bloco podem conter outros elementos de bloco ou em linha, mas elementos em linha não podem conter elementos de bloco. Essa diferença é fundamental para a estruturação e layout de páginas da web, afetando como os elementos são posicionados e exibidos no navegador
